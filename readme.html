xil

xil consta de tres partes:

  • un emulador de browser
  • un archivo .jpg de dimensiones de 1024x678px y un audio .wav con una duración de 64 segundos.
  • un ejecutable escrito en javascript
 

La aplicación está diseñada de tal manera que la grabación y la imagen se modifican irreversiblemente, a tal grado que después de n veces la pieza queda irreconozible, en otras palabras se deteriora permanentente.
